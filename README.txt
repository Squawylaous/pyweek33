This game was made in Python 3.6.6 and Pygame 1.9.3, but it should be compatible with any later versions.
Run main.py to run the game.

The goal of the game is to get to the end of each maze without letting your evil twin get to the end of thier maze first.
You are the gray square on the right, and your evil twin is the red square on the left. The white square is the end of the maze.
Whenever you move, your evil twin will move in the same direction. When you move, you move in that direction until you hit a wall.

Controls:
- WASD or arrow keys to move and navigate menus
- Space or Enter to select an option
- r to restart the level
- Esc to return to the menu (or quit game if already at menu)
